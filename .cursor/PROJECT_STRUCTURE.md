## Функциональность

### 1. Главная страница (`/`)
**Табы:**
- **Рекрутеры** - список всех рекрутеров с поиском
- **Вакансии** - список всех вакансий с поиском

**FAB (Floating Action Button):**
- Появляется при скролле
- На табе "Рекрутеры" - открывает форму создания рекрутера
- На табе "Вакансии" - открывает форму создания вакансии

**Таб Рекрутеры:**
- Поиск по рекрутерам (имя, email)
- Scrollable список рекрутеров
- Клик на рекрутера открывает детальную страницу

**Таб Вакансии:**
- Поиск по вакансиям (название, описание, локация)
- Scrollable список вакансий
- Клик на вакансию открывает drawer для просмотра/редактирования

### 2. Детальная страница рекрутера (`/recruiter/:id`)
- Информация о рекрутере
- Выбор компании из списка (Select компонент)
- Список вакансий выбранной компании
- Клик на вакансию открывает drawer

### 3. Drawers

**Drawer создания рекрутера:**
- Форма с полями: ФИО, Email, Телефон
- Валидация полей
- Заглушка для API (готово к интеграции с tanstack/query)

**Drawer создания вакансии:**
- Форма с полями: название, описание, компания, зарплата, локация, тип занятости, требования, обязанности
- Валидация полей
- Заглушка для API (готово к интеграции с tanstack/query)

**Drawer просмотра/редактирования вакансии:**
- Режим просмотра: полная информация о вакансии
- Режим редактирования: форма с возможностью изменения всех полей
- Переключение между режимами

## Технологии

- **React 19** - UI библиотека
- **TypeScript** - типизация
- **Mantine 8** - UI компоненты (@mantine/core, @mantine/hooks, @mantine/form, @mantine/notifications)
- **React Router 7** - роутинг
- **Vite** - сборщик
- **FSD** - архитектура
- **@tabler/icons-react** - иконки

## Готовность к интеграции с API

Все формы создания готовы к интеграции с **TanStack Query**:
- В компонентах есть комментарии TODO с примерами использования
- Структура данных соответствует TypeScript типам
- Обработка успешных операций через notifications
- Управление состоянием форм через @mantine/form
